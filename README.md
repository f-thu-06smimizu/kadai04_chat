# ２ちゃんねるっぽいやつ (リアルタイム公開チャット)

## 制作アプリの概要
FFirebase Realtime Databaseを用いた、匿名で利用可能なスレッド型掲示板アプリ。
リアルタイムメッセージングの基礎理解を目的としました。

---

## 工夫した点・こだわった点
* ローカル環境で発生したJavaScriptの import 構文のエラーを解決するため、すべての import 文を廃止し、**FirebaseのCDN（互換版SDK）**を直接HTMLで読み込む形式に切り替えることで、動作環境の制約を回避しました。
